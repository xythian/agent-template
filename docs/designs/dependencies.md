## Proposing New Dependencies

To propose a new dependency:

1. Create an RFC or discuss in an existing RFC
2. Document:
   - What the dependency is
   - Why it's needed
   - Alternatives considered
   - Maintenance/security considerations
3. Get explicit approval from a human maintainer
4. Update this document with the approved dependency
